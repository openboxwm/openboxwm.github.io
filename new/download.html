Openbox is available in most Linux distributions

To install Openbox on your Linux computer, run the following command (choose the one appropriate for your Linux distribution):

Ubuntu: gksu aptitude install openbox obconf

Fedora: gksu yum install openbox obconf

Arch: sudo pacman -S openbox obconf

Debian: sudo aptitude install openbox obconf

Gentoo: sudo emerge openbox obconf

Foresight: sudo conary update group-openbox

PcLinuxOS: sudo apt-get install openbox obconf

You can run these commands in a terminal, or through a launcher such as provided in GNOME by pressing Alt-F2.
